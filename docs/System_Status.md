## Recent Updates (v2.2.0)

### 1. Performance Optimizations
- **Rate Limiting**: Token bucket algorithm with per-IP tracking (100 req/min standard, 10 req/min auth)
- **Response Compression**: Gzip middleware for 30-50% bandwidth reduction
- **Database Pooling**: Connection pooling with pre-ping and automatic recycling
- **Caching Service**: In-memory cache with TTL support and cache invalidation helpers
- **Frontend Chunking**: Smart code splitting for better load times and caching

### 2. Security Enhancements
- **CSP Headers**: Comprehensive Content Security Policy
- **HSTS**: HTTP Strict Transport Security enabled
- **XSS Protection**: Enhanced headers for cross-site scripting prevention
- **Audit Logging**: 30+ event types for HIPAA/GDPR compliance

### 3. Dashboard Improvements
- **Doctor Dashboard**: Patient queue, clinical alerts, quick stats bar
- **Patient Dashboard**: Appointments, medication tracker, health goals
- **Lab Dashboard**: Stats grid, test categories, AI interpretation, critical alerts

### 4. New API Endpoints
- `/api/system/health/detailed` - Full system health check
- `/api/system/metrics` - Database counts, memory, CPU usage
- `/api/system/cache/stats` - Cache hit rates
- `/api/audit/logs` - Audit log queries with filters
- `/api/audit/export` - CSV/JSON export for compliance

### 6. Radiology Module (Active)
- **Dashboard**: PACS-like worklist, filtering by modality/status
- **AI Integration**: Auto-drafting of reports, finding detection
- **Backend**: Dedicated `ImagingStudy` models and CRUD API
- **Service**: Google Gemini integration for radiology insights
